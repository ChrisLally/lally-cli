# @chris-lally/cli

CLI for `@chris-lally/cli`.

> This README is auto-generated by `lally repo readme --target cli`.

## Install

### Prerequisites

- Node.js 20+
- npm 10+ (or pnpm/yarn/bun)
- git (for sync workflows)

```bash
pnpm add -D @chris-lally/cli
```

or run without install:

```bash
pnpm dlx @chris-lally/cli@alpha --help
```

### `lally --help`

```text
lally

  lally fumadocs <init|section|page-shell|sidebar-history|check|clean|layout|generate> [options]

Examples:

  lally git sync doctor --target statements
```
